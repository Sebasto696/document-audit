# SPEC.md - Landing Page E-commerce de Sneakers

## 1. Project Overview

- **Project Name**: SoleVault - Premium Sneakers
- **Project Type**: Landing Page / E-commerce
- **Core Functionality**: Showcase de sneakers premium con diseño urbano moderno, sección de productos destacados, características de marca, y llamado a la acción para compra
- **Target Users**: Jóvenes adultos (18-35) interesados en moda urbana y sneakers de edición limitada

## 2. UI/UX Specification

### Layout Structure

**Secciones:**
1. **Navigation Bar** - Logo, menú links, icono de carrito
2. **Hero Section** - Imagen grande de sneaker, headline, CTA
3. **Featured Products** - Grid de 3-4 productos destacados
4. **Features Section** - 3 iconos con beneficios de la marca
5. **Testimonials** - Carrusel de testimonios de clientes
6. **Newsletter CTA** - Suscripción con email
7. **Footer** - Links, redes sociales, copyright

**Responsive Breakpoints:**
- Mobile: < 768px (1 columna)
- Tablet: 768px - 1024px (2 columnas)
- Desktop: > 1024px (3-4 columnas)

### Visual Design

**Color Palette:**
- Background Primary: `#0D0D0D` (negro profundo)
- Background Secondary: `#1A1A1A` (gris oscuro)
- Accent Primary: `#FF4D4D` (rojo vibrante)
- Accent Secondary: `#00D9FF` (cyan electrico)
- Text Primary: `#FFFFFF` (blanco)
- Text Secondary: `#A0A0A0` (gris claro)
- Card Background: `#141414`

**Typography:**
- Headings: "Bebas Neue" (bold, uppercase)
- Body: "DM Sans" (regular/medium)
- Hero Title: 72px desktop / 40px mobile
- Section Titles: 48px desktop / 32px mobile
- Body Text: 16px

**Spacing System:**
- Section padding: 80px vertical (desktop), 40px (mobile)
- Container max-width: 1200px
- Card padding: 24px
- Gap entre elementos: 24px

**Visual Effects:**
- Hover en cards: escala 1.02, shadow rojo
- Gradient en hero: diagonal de #FF4D4D a #00D9FF
- Efecto glassmorphism en navbar
- Animación de entrada staggered en productos

### Components

**Navbar:**
- Fijo en scroll
- Fondo glassmorphism con blur
- Logo a la izquierda
- Links centrado: Home, Colección, Acerca de, Contacto
- Icono de carrito a la derecha

**Hero:**
- Half-half layout (texto izq, imagen der)
- Badge "Nueva Colección 2026"
- Headline grande con gradient text
- Subtítulo descriptivo
- 2 botones: "Ver Colección" (rojo) y "Más Información" (outline)

**Product Card:**
- Imagen del sneaker (aspect ratio 1:1)
- Nombre del producto
- Precio
- Badge de descuento (si aplica)
- Hover: imagen zoom slight, botón "Añadir" aparece

**Features:**
- 3 columnas con ícono, título, descripción
- Íconos: Envío gratis, Autenticidad garantizada, Devoluciones
- Fondo con grid pattern sutil

**Testimonials:**
- Cards horizontales
- Foto del cliente, nombre, valoración (estrellas), comentario
- Diseño quote con comillas decorativas

**Newsletter:**
- Fondo con gradient
- Input de email + botón de suscripción
- Texto motivador

**Footer:**
- 4 columnas: Logo/descripción, Links rápidos, Shop, Contacto
- Iconos de redes sociales
- Copyright

## 3. Functionality Specification

### Core Features
- Navegación suave entre secciones (scroll)
- Hover effects interactivos en todos los elementos clickeables
- Diseño responsive que se adapta a todos los tamaños
- Animaciones de entrada al hacer scroll (fade-in)

### User Interactions
- Click en nav links = scroll suave a sección
- Hover en productos = efecto visual
- Hover en botones = cambio de color/sombra
- Input de newsletter con validación visual

### Data Handling
- Productos hardcodeados en array (simulando backend)
- No requiere backend real para esta landing page
